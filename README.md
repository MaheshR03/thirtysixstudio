## INSTALLATION GUIDE

```shell
git clone https://github.com/MaheshR03/thirtysixstudio
```
```shell
npx create-vite@latest
```
```shell
npm i
```
```shell
npm install -D tailwindcss@3 postcss autoprefixer
```
```shell
npx tailwindcss init -p
```
```shell
npm i gsap
```
```shell
npm i @gsap/react
```
```shell
npm install locomotive-scroll@beta
```
